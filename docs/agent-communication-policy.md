# Communication agents

Canaux :

- Kanban : taches durables, commentaires, validations et handoffs.
- Swarm : coordination live de profiles.
- MCP memoire : connaissances durables par `project_id`.
- Rapports : preuves, risques, decisions et prochaine action.
- `delegate_task` : sous-taches temporaires courtes.

Chaque handoff identifie projet, tache, agent suivant, livrable, risques et
validation humaine requise.
