# 07 — Drawdown

## Définition
Baisse depuis un pic d'équité jusqu'à un creux. Mesure la douleur d'une stratégie.

## Pourquoi ça compte
Un drawdown profond peut être irrécupérable (perdre 50 % exige +100 % pour
revenir). La maîtrise du drawdown > la maximisation du gain.

## Règles module
- Perte max / jour 2 %, / semaine 5 % -> au-delà : NO_TRADE.
- Surveiller le drawdown cumulé dans le journal.

## En backtest
Comparer le drawdown max entre stratégies/paramètres. Un PnL élevé avec un
drawdown énorme = stratégie fragile.
