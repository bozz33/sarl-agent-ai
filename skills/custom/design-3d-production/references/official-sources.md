# Sources officielles

Priorité: documentation versionnée, spécification, puis notes de version.

- Blender Manual: https://docs.blender.org/manual/en/latest/
- Blender Python API: https://docs.blender.org/api/current/
- Blender release notes: https://developer.blender.org/docs/release_notes/
- FreeCAD documentation: https://wiki.freecad.org/
- FreeCAD source and releases: https://github.com/FreeCAD/FreeCAD
- Khronos glTF specification: https://registry.khronos.org/glTF/
- OpenUSD documentation: https://openusd.org/release/
- MaterialX specification: https://materialx.org/
- OpenColorIO documentation: https://opencolorio.readthedocs.io/

Pour chaque source, enregistrer URL, date d’accès, version, section et conclusion.
Ne pas extrapoler une documentation `latest` vers version installée sans contrôle.
