# Cost Control

1. Utiliser modele le moins couteux capable de reussir.
2. Reserver Claude, GPT et Codex aux taches avancees ou critiques.
3. Journaliser chaque appel premium dans `reports/costs/`.
4. Ne pas utiliser premium pour resume, extraction ou reformulation simple.
5. Fallback technique gere panne ou quota; escalade gere complexite ou risque.
