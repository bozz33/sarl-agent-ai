# Governance Audit

Verifier :

- projet correctement identifie ;
- absence de melange inter-projets ;
- classification de criticite ;
- validation humaine pour actions critiques ;
- usage premium justifie ;
- rapport final present ;
- memoire correctement typee ;
- secrets absents des sorties ;
- backup et rollback presents avant changement important.

Produire constat, preuve, severite, correction et proprietaire.
